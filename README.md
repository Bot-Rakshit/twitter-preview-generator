# Twitter Preview Generator

A React application that creates pixel-perfect Twitter/X tweet mockups. Features include:

- Custom profile name and handle
- Post content editing
- Profile picture and post image upload
- Timestamp and view count customization
- High-quality image export using dom-to-image

## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

```bash
npm install
npm start
```

The app will run on http://localhost:3000

## Features

- **Live Preview**: See changes in real-time as you edit
- **Image Upload**: Support for profile pictures and post images
- **High-Quality Export**: Downloads rendered at 2x scale for crisp images
- **Verified Badge**: Toggle verification status
- **Custom Styling**: Dark mode Twitter-like interface

## Built With

- React 18
- Tailwind CSS
- Lucide React Icons
- dom-to-image

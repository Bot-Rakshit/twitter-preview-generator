A React application that creates pixel-perfect Twitter/X tweet mockups. Built with modern web technologies for creators, marketers, and developers who need high-quality tweet screenshots.

![Twitter Preview Generator](public/og-image.png)

## Features

- **Live Preview**: See changes in real-time as you edit
- **Custom Profiles**: Set display name, username, avatar, and verification badge
- **Post Content**: Write unlimited-length tweets with full formatting support
- **Image Upload**: Add profile pictures and post images
- **Timestamp & Views**: Customize engagement metrics
- **High-Quality Export**: Downloads rendered at 2x scale for crisp images
- **Dark Mode Interface**: Twitter-like dark theme for authentic previews
- **Responsive Design**: Works on desktop and mobile devices

## Quick Start

```bash
# Clone the repository
git clone https://github.com/Bot-Rakshit/twitter-preview-generator.git

# Navigate to the project
cd twitter-preview-generator

# Install dependencies
npm install

# Start the development server
npm start
```

The app will run on http://localhost:3000

## Usage

1. **Edit Profile**: Enter the display name and username (@handle)
2. **Write Content**: Type your tweet in the content area
3. **Upload Images**: Add a profile picture and/or post image
4. **Set Metadata**: Adjust timestamp and view count
5. **Download**: Click "Download Image" to save your mockup
